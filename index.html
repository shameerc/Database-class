Database class by shameerc

PHP class for interfacing database with a wrapper class which will allow to add more underlying db connectors or adapters.

This PHP package includes a Mysqli database driver class and a wrapper class which will allow to add more db drivers. The wrapper class provides two methods to connect to the database, viz factory and singleton. Factory method will allow the application to have multiple database connection, where as the singleton connection will create only one connection using the default connection string.
